# Steps to run the app
1. Clone the repo
2. Navigate to the directory and run npm install
3. After the installation, run npm start
4. The app should be running on http://localhost:3000

routes/index.js - Adds the users to the firebase database
routes/users.js - Perform function returns all nodes in the database
                  Update function updates the particular node

In the front end, sign up with your name. On signing up, all the nodes are displayed which can be refreshed by clicking on the refresh icon. To update the node value, just edit the text and on change the value gets updated.
